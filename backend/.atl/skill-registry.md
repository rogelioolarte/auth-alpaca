## Project Conventions (from GEMINI.md)

### Overview

This is a backend project for a Multiple Auth API using OAuth2 and JWT, built with Java and Spring Boot, persisting data to PostgreSQL.

### Technology Stack

- **Language**: Java
- **Framework**: Spring Boot
- **Build Tool**: Maven
- **Database**: PostgreSQL
- **ORM/Data Access**: Spring Data JPA (inferred)
- **Web Framework**: Spring Web MVC for REST APIs (inferred)
- **Testing**: JUnit, Mockito, Spring Boot Test
- **Other Libraries**: Lombok, Slf4j (inferred)

### Architecture

Adheres to a layered architecture with API, Service, Domain, and Persistence layers.

### Project Structure

Standard Maven layout with `/src/main/java`, `/src/main/resources`, `/src/test/java`, and `/src/test/resources`.

### Current Status

The main application source code (`/src/main/java`) is refactored and in a good state. However, the test suite (`/src/test/java`) **still needs to be updated** to align with recent changes in the main code.

### Future Work

Prioritize updating the test suite in `/src/test/java` for comprehensive coverage.
