Multiple Auth API

Spring Boot application that implements multiple authentication methods using OAuth2 and JWT, with support for standard login, Google OAuth2, and session management.

Stack

  • Spring Boot 4 / Java 25
  • Spring Security 6 / OAuth2 Client
  • Spring Data JPA + PostgreSQL
  • JWT (access + refresh tokens) with ES256 (ECDSA P-256)
  • PKCE flow for OAuth2 authorization code grant
